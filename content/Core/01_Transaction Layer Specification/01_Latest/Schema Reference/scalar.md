Scalar
===
>An object representing a scalar quantity. Extension allowed only to the limit of vector

### Schema Definition

|**Field**|**Type**|**Description**|
|---------|--------|---------------|
|type|string|. Allowed values:CONSTANT, VARIABLE
|value|number|
|estimated_value|number|
|computed_value|number|
|range.min|number|
|range.max|number|
|unit|string|
